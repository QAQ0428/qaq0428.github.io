JS [a, b]的随机整数

2024年3月22日 22点56分

场景

在前端开发中, 有许多用到随机数的场景. 但是JS中的随机数函数Math.random返回的是0-1的随机浮点数. 因此我们需要做一些运算(非常简单的).

思路

想要实现[a, b]间的随机整数, 其实就是一个简单的数学问题: 已知$x\in\left[ 0, 1 \right)$, 想要变形得到一个$f\left(x\right)\in[a, b]\cap\Z$.

过程

$$ x = \operatorname{Math.random}\left(\right)\\ 0 \leq x < 1 $$ 同时$\times \left(b - a + 1\right)$, 就得到 $$0 \leq \left(b - a + 1\right)x < b - a + 1$$ 同时+a就得到 $$a \leq \left(b - a + 1\right)x + a< b + 1$$ 因为想要得到一个整数, 所以对$\left(b - a + 1\right)x + a$进行向下取整. 最终, $\lfloor\left(b - a + 1\right)x + a\rfloor\in\left[a, b\right]\cap\Z$

JS代码

                const randInt = (a, b) => Math.floor((b - a + 1) * Math.random() + a)
